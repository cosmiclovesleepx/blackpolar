# Imágenes - TLM App

Esta carpeta contiene todas las imágenes utilizadas en la aplicación TLM.

## Estructura Recomendada

```
image/
├── course-thumbnails/  - Miniaturas de cursos
├── instructor-avatars/ - Avatares de instructores
├── banners/            - Banners de cursos
└── icons/              - Iconos de categorías
```

## Formatos Soportados

- **JPEG** (.jpg, .jpeg)
- **PNG** (.png)
- **WebP** (.webp)
- **SVG** (.svg)

## Tamaños Recomendados

- **Course Thumbnails**: 300x200px (3:2 ratio)
- **Avatars**: 150x150px (1:1 square)
- **Banners**: 1200x300px (4:1 ratio)
- **Icons**: 64x64px (vectorial SVG preferido)

## Buenas Prácticas

1. Usar iconos SVG para mejor escalabilidad
2. Proporcionar versiones 2x para retina displays
3. Optimizar para carga rápida en plataforma educativa
4. Mantener consistencia visual entre elementos
5. Documentar atribuciones de imágenes CC0 o con licencia
