# Imágenes - Portfolios App

Esta carpeta contiene todas las imágenes utilizadas en la aplicación Portfolios.

## Estructura Recomendada

```
image/
├── portfolio-1/     - Imágenes del portafolio 1
├── portfolio-2/     - Imágenes del portafolio 2
├── portfolio-3/     - Imágenes del portafolio 3
├── thumbnails/     - Thumbnails para galería
└── backgrounds/     - Imágenes de fondo
```

## Formatos Soportados

- **JPEG** (.jpg, .jpeg)
- **PNG** (.png)
- **WebP** (.webp)
- **SVG** (.svg)

## Optimización de Galerías

Para galerías de portafolios, se recomienda:

1. **Imagen grande** (1200x800px) - Para vista principal
2. **Thumbnail** (300x200px) - Para vista previa
3. **WebP optimizado** - Ideal para web

## Buenas Prácticas

1. Mantener aspectos consistentes entre galerías
2. Usar lazy loading para mejora de rendimiento
3. Proporcionar versiones mobile-friendly
4. Añadir descripciones alt para accesibilidad
5. Organizar por carpetas según proyectos
